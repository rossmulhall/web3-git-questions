Review 3 different GUI Git clients

Git Kraken

Git Kraken is a heavily GUI based Git client providing integration for OSX, Windows and Linux

Git Box

Git Box is a powerful tool for working with Git as it provides users with the ability to simply click to do even the most complex of tasks.

Fork

As it says on their website Fork is both ‘fast and friendly’ for the user. It has a well organised GUI which allows the user to see the effect of everything they do to their repository. It also gives the user Github notifications which is a nice added feature
